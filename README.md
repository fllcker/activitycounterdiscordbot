<h1 align="center">ActivityCounterDiscordBot</h1>

Discord bot for viewing user activity in games, voice channels, etc.

Example:
```
 /activities
 [Voice channel name]
 precision#5206 - 20(min)

 [osu!]
 precision#5206 - 15(min)
 ```

# Stack of techologies
- dotnet 6
- DISCORD.NET
